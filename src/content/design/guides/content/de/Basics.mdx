Gravity-UI enthält eine Sammlung von Bibliotheken und Komponenten zur Erstellung funktionaler Benutzeroberflächen. Das Designsystem unterstützt die Arbeit mit mehreren Themes: Es gibt ein **helles**, ein **dunkles** und deren Versionen mit **erhöhtem Kontrast**. Das Aussehen der Komponenten kann für jedes dieser Themes flexibel angepasst werden.

**Farbe**

Das Designsystem enthält 4 Themes, die auf einem zweischichtigen Farbmodell basieren. Der Theming-Mechanismus ist für die flexible Farbanpassung und die Unterstützung der Identität von Marken mit unterschiedlichen Anforderungen konzipiert.

**Typografie**

Das Typografiesystem besteht aus Stilen für Fließtext, Akzidenz und Code (Monospace-Schriftart). Sie können sowohl die Standard-Schriftarten verwenden als auch eigene festlegen.

**Abrundungsradien von Steuerelementen**

Ein kleiner, aber auffälliger Teil des visuellen Stils sind die Abrundungsradien der Ecken. Dafür wurde ein Set von Variablen für die Radien erstellt. Diese Variablen werden in verschiedenen Steuerelementen verwendet. Um den Radius in Steuerelementen anzupassen, genügt es, die Werte der Variablen zu ändern.

**Erstellung und Arbeit mit Themes**

Unabhängig von der Marke und ihrer visuellen Sprache läuft alles auf eine Reihe von Konstanten hinaus, die bearbeitet werden müssen und ein einheitliches Set im Code und in der Figma-Bibliothek verwendet werden muss.
