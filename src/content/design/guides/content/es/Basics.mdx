Gravity-UI incluye un conjunto de bibliotecas y componentes para crear interfaces funcionales. El sistema de diseño admite el trabajo con varios temas: hay una versión **clara**, una **oscura** y sus versiones con **alto contraste**. La apariencia de los componentes se puede personalizar de forma flexible para cada uno de estos temas.

**Color**

El sistema de diseño contiene 4 temas basados en un modelo de color de dos capas. El mecanismo de tematización está diseñado para una personalización flexible del color y para admitir la identidad de marcas con diferentes conjuntos de requisitos.

**Tipografía**

El sistema tipográfico consta de estilos para texto de cuerpo, texto de acento y código (fuente monoespaciada). Puede usar las fuentes predeterminadas o especificar las suyas.

**Radios de redondeo de controles**

Una parte pequeña pero notable del estilo visual son los radios de redondeo de las esquinas. Para esto, se ha creado un conjunto de variables para los radios. Estas variables se utilizan en diferentes controles. Para ajustar el radio en los controles, basta con cambiar los valores de las variables.

**Creación y trabajo con temas**

Independientemente de la marca y su lenguaje visual, todo se reduce a un conjunto de constantes que deben poder editarse y utilizar un conjunto unificado en el código y en la biblioteca de Figma.
