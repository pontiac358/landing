- In cases where it's useful to customize the number of rows displayed.
- To fix the size of blocks within a page, so that the automatic loading of content at the top doesn't constantly shift the content at the bottom.

## Structure

![IMG - 18.png](/static/images/design/Pagination/paginator-0005.png)

1. **Current page** â€” displays the sequential number of the page being shown.
2. **Previous page** â€” opens the previous page.
3. **Next page** â€” opens the next page.
4. **Last content page** â€” shown if the total content count is known and can be immediately divided into pages. Clicking it opens the last page.
5. **Overflow icon** â€” appears if there are more than 4 pages.
6. **Forward/backward navigators** â€” switch pages sequentially.
7. **Go to first page** â€” opens the first page of content.
8. **Go to specific page** â€” by entering the page number and pressing Enter, opens the specified page.
9. **Number of items per page** â€” a switcher for the number of items displayed on one page.

It's not necessary to use all structural elements together. Available configurations are listed in the "Modifications" section.
