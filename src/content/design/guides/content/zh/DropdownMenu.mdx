The component allows the user to select a parameter or action from a contextual list. A button with a three-dot icon is typically used to invoke the dropdown list. The icon can be reassigned to any other, and switching between list items can be controlled with the keyboard.

# Size

![42.png](/static/images/design/DropdownMenu/42.png)

**Icon**

By default, the context menu trigger icon has 4 built-in sizes:

`S` (24 x 24 px), `M` (28 x 28 px), `L` (36 x 36 px), and `XL` (44 x 44 px).

![Group 2087326002.png](/static/images/design/DropdownMenu/Group_2087326002.png)

**Component Width**

In cases where a menu item is longer than the object that triggers it, the menu will expand to the width of the longest item in the list. The maximum component width is 280 px. If the menu item text exceeds this width, it will be truncated with an ellipsis.

# Applicability

![43.png](/static/images/design/DropdownMenu/43.png)

**Customizable Trigger Icon**

The context menu trigger icon can be resized or replaced with any other icon. By default, the component uses a three-dot button with an icon size of 16 px.

![44.png](/static/images/design/DropdownMenu/44.png)

**Optional Menu Item Icon**

Context menu items can include icons for better understanding. By default, the icon size in the context menu is 16x16 px.

![41.png](/static/images/design/DropdownMenu/41%201.png)

**Nested Dropdown Submenus**

Context menu items can be nested within each other. By default, nesting is supported up to 3 levels deep.

![46.png](/static/images/design/DropdownMenu/46.png)

**Placement**

默认情况下，下拉菜单会在触发它的元素下方打开，菜单图标位于右侧。如果界面不允许在触发器图标下方渲染下拉列表，则列表会在图标上方展开。

![47.png](/static/images/design/DropdownMenu/47.png)

**关闭上下文菜单**

可以通过点击菜单外部或按 Esc 键来关闭上下文菜单。滚动页面时，菜单会隐藏（可选，此行为可以禁用）。

# 建议

当您有至少两个选项时，可以使用下拉菜单。最佳选项数量为 5 到 10 个。

在组织菜单项时，请遵循逻辑顺序，将最常用和最受欢迎的选项放在顶部。测试不同的菜单项，并根据需要进行调整。
