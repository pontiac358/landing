Elemento clicável com função de navegação. Ao ser clicado, abre outra página de um serviço externo ou interno. Mais raramente, move o usuário para outra parte da página.

Podem ser de três tipos: `normal` (destaque), `primary` (preto), `secondary` (cinza).

**Tipo `normal`**

O padrão de uso mais familiar e estabelecido para os usuários. É aplicado para separar visualmente o elemento dentro de um texto, tabela e como parte da navegação. Também é aplicado em páginas de erro e estados vazios.

**Tipo `primary`**

Usado quando é intuitivamente claro para o usuário que o elemento pode ser interagido, mas o uso de links com cores de destaque sobrecarregaria a interface e quebraria o equilíbrio dos elementos na página.

**Tipo `secondary`**

Assim como os links `primary`, são usados quando é claro para o usuário que o elemento pode ser interagido, e há a tarefa de não competir com os destaques na página. Mais frequentemente usado em breadcrumbs ou ao exibir atributos secundários.

## **Estados**

Links sempre têm um estado padrão e um estado de hover.

![Frame 507123500.jpg](/static/images/design/Links/Frame_507123500.png)

Se necessário, a marcação para links visitados pode ser ativada.

## **Podem ser aplicados como:**

- elementos isolados;
- parte de um texto unificado.

Podem ter um ícone como indicador de transição para um recurso externo.

![Frame 507123501.jpg](/static/images/design/Links/Frame_507123501.png)

## Tamanho

A fonte do texto para links deve corresponder à hierarquia geral na página e ser proporcional ao texto circundante.

## Recomendações

O texto do link deve ser compreensível para o usuário mesmo quando separado do restante do texto e conter informações sobre para onde o link redireciona (nome do serviço, título ou descrição da seção).
