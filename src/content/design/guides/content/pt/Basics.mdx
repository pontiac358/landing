O Gravity-UI contém um conjunto de bibliotecas e componentes para criar interfaces funcionais. O sistema de design suporta o trabalho com vários temas: há **claro**, **escuro** e suas versões com **alto contraste**. A aparência dos componentes pode ser configurada de forma flexível para cada um desses temas.

**Cor**

O sistema de design contém 4 temas, baseados em um modelo de cores de duas camadas. O mecanismo de tematização foi projetado para configuração flexível de cores e suporte à identidade de marcas com diferentes conjuntos de requisitos.

**Tipografia**

O sistema de tipografia consiste em estilos para texto corrido, texto de destaque e código (fonte monoespaçada). Você pode usar as fontes padrão ou definir as suas próprias.

**Raios de arredondamento de controles**

Uma parte pequena, mas notável, do estilo visual são os raios de arredondamento dos cantos. Para isso, foi criado um conjunto de variáveis para os raios. Essas variáveis são usadas em diferentes controles. Para ajustar o raio nos controles, basta alterar os valores das variáveis.

**Criação e trabalho com temas**

Independentemente da marca e de sua linguagem visual, tudo se resume a um conjunto de constantes que precisam ser editáveis e usadas em um conjunto unificado no código e na biblioteca do Figma.
