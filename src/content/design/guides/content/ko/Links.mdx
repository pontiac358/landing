클릭 가능한 네비게이션 요소입니다. 클릭 시 서비스 내부 또는 외부의 다른 페이지로 이동합니다. 드물게 페이지의 다른 섹션으로 이동하는 데 사용됩니다.

세 가지 유형이 있습니다: `normal` (강조), `primary` (검정), `secondary` (회색).

**`normal` 유형**

사용자에게 가장 익숙하고 일반적인 사용 패턴입니다. 텍스트, 테이블 내에서 요소를 시각적으로 구분하거나 네비게이션의 일부로 사용됩니다. 오류 페이지 및 제로 상태 페이지에서도 사용됩니다.

**`primary` 유형**

사용자가 요소와 상호작용할 수 있다는 것을 직관적으로 알 수 있지만, 강조 색상의 링크를 사용하면 인터페이스가 과부하되고 페이지 요소 간의 균형이 깨질 때 사용됩니다.

**`secondary` 유형**

`primary` 링크와 마찬가지로, 사용자가 요소와 상호작용할 수 있다는 것을 알 수 있고 페이지의 강조 요소와 충돌하지 않도록 해야 할 때 사용됩니다. 주로 브레드크럼이나 보조 속성을 표시할 때 사용됩니다.

링크는 항상 기본 상태와 호버 상태를 가집니다.

## **사용 방식:**

- 독립적인 요소로 사용;
- 텍스트의 일부로 사용.

외부 리소스로의 이동을 나타내는 지표로 아이콘을 포함할 수 있습니다.

![Frame 507123501.jpg](/static/images/design/Links/Frame_507123501.png)

## 크기

링크 텍스트의 글꼴은 페이지의 전반적인 계층 구조와 일치해야 하며 주변 텍스트와 비례해야 합니다.

## 권장 사항

링크 텍스트는 나머지 텍스트와 분리되어도 사용자에게 명확해야 하며, 링크가 어디로 연결되는지에 대한 정보(서비스 이름, 섹션 제목 또는 설명)를 포함해야 합니다.
