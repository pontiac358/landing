Gravity-UI propose un ensemble de bibliothèques et de composants pour créer des interfaces fonctionnelles. Le système de design prend en charge plusieurs thèmes : il existe un thème **clair**, un thème **sombre** et leurs versions à **contraste élevé**. L'apparence des composants peut être personnalisée de manière flexible pour chacun de ces thèmes.

**Couleur**

Le système de design contient 4 thèmes basés sur un modèle de couleur à deux couches. Le mécanisme de thématisation est conçu pour une personnalisation flexible des couleurs et pour prendre en charge l'identité de marques avec des ensembles d'exigences variés.

**Typographie**

Le système typographique se compose de styles pour le texte courant, les éléments d'accentuation et le code (police monospace). Vous pouvez utiliser les polices par défaut ou définir les vôtres.

**Rayons d'arrondi des contrôles**

Une partie petite mais notable du style visuel réside dans les rayons d'arrondi des coins. Pour cela, un ensemble de variables pour les rayons a été créé. Ces variables sont utilisées dans différents contrôles. Pour ajuster le rayon dans les contrôles, il suffit de modifier les valeurs des variables.

**Création et utilisation d'un thème**

Quel que soit le langage visuel d'une marque, tout se résume à un ensemble de constantes qu'il faut pouvoir éditer et utiliser un ensemble unique dans le code et dans la bibliothèque Figma.
