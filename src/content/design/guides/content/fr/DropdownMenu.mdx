Le composant permet à l'utilisateur de sélectionner une option ou une action dans une liste contextuelle. Pour ouvrir la liste déroulante, on utilise généralement un bouton avec une icône représentant trois points. L'icône peut être remplacée par n'importe quelle autre, et la navigation entre les éléments de la liste est possible au clavier.

# Taille

![42.png](/static/images/design/DropdownMenu/42.png)

**Icône**

Par défaut, l'icône d'appel du menu contextuel dispose de 4 tailles intégrées :

`S` (24 x 24 px), `M` (28 x 28 px), `L` (36 x 36 px) et `XL` (44 x 44 px).

![Group 2087326002.png](/static/images/design/DropdownMenu/Group_2087326002.png)

**Largeur du composant**

Lorsque le texte d'un élément de menu est plus long que l'objet qui le déclenche, le menu s'agrandira pour s'adapter à la largeur de l'élément le plus long de la liste. La largeur maximale du composant est de 280 px. Si le texte d'un élément de menu dépasse cette largeur, il sera tronqué avec des points de suspension.

# Applicabilité

![43.png](/static/images/design/DropdownMenu/43.png)

**Icône d'appel personnalisable**

L'icône d'appel du menu contextuel peut être redimensionnée ou remplacée par n'importe quelle autre. Par défaut, le composant utilise un bouton représentant trois points avec une icône de 16 px.

![44.png](/static/images/design/DropdownMenu/44.png)

**Icône optionnelle pour les éléments de menu**

Les éléments du menu contextuel peuvent inclure des icônes pour améliorer la compréhension. Par défaut, la taille de l'icône dans le menu contextuel est de 16x16 px.

![41.png](/static/images/design/DropdownMenu/41%201.png)

**Sous-menus déroulants imbriqués**

Les éléments du menu contextuel peuvent être imbriqués les uns dans les autres. Par défaut, l'imbrication est prise en charge jusqu'à 3 niveaux de profondeur.

![46.png](/static/images/design/DropdownMenu/46.png)

**Positionnement**

Par défaut, le menu déroulant s'ouvre sous l'élément qui le déclenche, et l'icône du menu se trouve dans le coin droit. S'il n'est pas possible de rendre la liste déroulante sous l'icône du déclencheur dans l'interface, la liste se déroule au-dessus de l'icône.

![47.png](/static/images/design/DropdownMenu/47.png)

**Fermeture du menu contextuel**

Le menu contextuel peut être fermé en cliquant en dehors de celui-ci ou en appuyant sur la touche Échap. Lorsque la page défile, le menu se masque (facultatif, ce comportement peut être désactivé).

# Recommandations

Les menus déroulants sont utilisés lorsque vous avez un choix d'au moins deux options. Le nombre optimal d'options est de 5 à 10.

Lors de l'organisation des éléments du menu, respectez un ordre logique, en plaçant les options les plus demandées et les plus populaires en haut. Testez différents éléments de menu et affinez-les si nécessaire.
